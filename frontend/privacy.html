Privacy Notice

Last updated: May 2026

What we collect

This site uses Microsoft Azure Application Insights solely to understand how the app is used in aggregate — for example, how many people visit each day and which features are used. No names, email addresses, or login credentials are collected because the site has no accounts.

What Application Insights records

  • Page views — which page was loaded and roughly when.
  • Browser & device type — e.g. Chrome on Android, Safari on iOS.
  • Country / region — derived from the anonymised IP address. The full IP address is not stored by Application Insights.
  • Performance timings — how long pages and API calls take to load.
  • Errors — JavaScript exceptions or failed network requests, so we can fix bugs.

Two first-party cookies are set by the SDK:

  • ai_user — a random identifier stored for up to 365 days, used to count distinct visitors (not tied to any personal identity).
  • ai_session — a session identifier, expires when you close the browser tab.

What we do not collect

  • No names, email addresses, or any account information (the site has no accounts).
  • No full IP addresses — Microsoft truncates them before storage.
  • No advertising or marketing tracking of any kind.
  • We do not sell, share, or monetise any data.

Data processor

Analytics data is processed by Microsoft Corporation under their Privacy Statement. Data is stored in the Azure region selected during resource provisioning and subject to Microsoft's data residency commitments.

Retention

Application Insights retains data for 90 days by default. No other copies are kept.
